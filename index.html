Cómo Publicar una Página Web

Pasos para publicar en GitHub Pages

  1. Crear un repositorio en GitHub:
    • Inicia sesión en tu cuenta de GitHub.
    • Haz clic en "New" para crear un nuevo repositorio.
    • Asigna un nombre, por ejemplo, mi-pagina-web.
    • Marca la opción "Public" para que sea público.
    • Opcional: añade un archivo README.
    • Haz clic en "Create repository".
  2. Preparar tu código HTML:
    • Asegúrate de tener tu archivo index.html listo.
    • Incluye otros archivos si es necesario (imágenes, CSS, JS).
  3. Subir tus archivos a GitHub:
    • En tu repositorio, haz clic en "Add file" > "Upload files".
    • Arrastra o selecciona tus archivos y confirma.
  4. Publicar con GitHub Pages:
    • Ve a "Settings" en tu repositorio.
    • Selecciona la pestaña "Pages".
    • En "Source", elige la rama "main" y la carpeta raíz (/).
    • Haz clic en "Save".
  5. Acceder a tu página web:
    • Espera unos minutos, y visita:
      https://tu-usuario.github.io/mi-pagina-web/
¡Ya tienes tu página web publicada en GitHub Pages!
